# KingsWorld - Executor Diario Otimo (Runs Perfect)

- Arquivo prescritivo baseado nas runs `perfect` do modo `paired8`.
- Cada dia mostra a melhor lista pratica de acoes para repetir a build vencedora.
- Aqui a prioridade ja vem convertida em execucao concreta.
- Todas as seeds deste arquivo sao vencedoras por design; use o `action_timelines` para comparar com as `lazy`.
- A coluna `Margem` mostra quando a build realmente encosta ou passa do corte 1500.
